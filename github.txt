Git hub

For the first time users
    git config --global user.name "your github username"
    git config --global user.email "your github email"
    
for 1st time
    git init = initialize or setup git in your project (F_)
    git add . = add/mark all your files
    git commit -m "your message" = to represent your changes
    git remote add origin <link> = to connect your machine to Github Servers
    git push origin master = upload to Github

For the rest of the time
    git add . = add/mark all your files
    git commit -m "your message" = to represent your changes 
    git push origin master = upload to Github
